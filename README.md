# PL_Ruby_Final_Project

## THIS IS FOR EDUCATIONAL PURPOSE ONLY

The Version of Ruby Used is 3.1.4 and Tk is 0.5.1

## Features

- **Add Ingredient**: Add new burger ingredients with name, quantity, and price
- **View All Ingredients**: Display a scrollable list of all ingredients with total cost
- **Update Ingredient**: Modify existing ingredient details via combobox selection
- **Delete Ingredient**: Remove ingredients from inventory with confirmation
- **Real-time Cost Calculation**: Automatic total cost calculation and display
- **Centered GUI Windows**: All windows are 900x500 and centered on screen
- **Responsive Layout**: Listbox and input fields expand to fit window size
- **Data Persistence**: Maintains ingredient list during session
